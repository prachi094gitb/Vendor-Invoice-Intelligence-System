# 📦 Vendor Invoice Intelligence System

An end-to-end Machine Learning application that automates invoice analysis by predicting freight costs and identifying invoices that require manual approval.

Built with Python, Machine Learning, and Streamlit, this system helps finance and procurement teams reduce manual effort, improve cost forecasting, and streamline invoice processing workflows.

---

## 🚀 Project Overview

Organizations process thousands of vendor invoices every month, making manual verification time-consuming and error-prone.

This project leverages Machine Learning to:

✅ Predict freight costs before invoice processing  
✅ Flag high-risk invoices requiring manual review  
✅ Reduce operational overhead  
✅ Improve financial decision-making

---

## 🎯 Problem Statement

Vendor invoice management often faces challenges such as:

- Unpredictable freight expenses
- Delayed invoice approvals
- Manual verification effort
- Financial leakage due to invoice anomalies
- Lack of predictive insights

The Vendor Invoice Intelligence System addresses these challenges using data-driven predictive models.

---

## ✨ Features

### 🚚 Freight Cost Prediction

Predict the expected freight charge based on invoice details.

**Benefits**
- Better budgeting
- Cost forecasting
- Freight anomaly detection
- Improved vendor negotiations

---

### 🚨 Invoice Approval Intelligence

Automatically classifies invoices into:

- ✅ Safe for Auto Approval
- ⚠️ Requires Manual Approval

**Benefits**
- Faster invoice processing
- Reduced risk exposure
- Improved compliance
- Lower manual workload
